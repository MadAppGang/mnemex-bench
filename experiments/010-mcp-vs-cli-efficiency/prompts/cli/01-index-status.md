Check the current index status of this codebase.

Run `mnemex status` via the Bash tool and report:
1. Whether the index is initialized (true/false)
2. How many files are indexed
3. Whether the index is fresh or stale
4. The index path

IMPORTANT: Only use the mnemex CLI via the Bash tool. Do NOT use MCP tools, Read, Grep, Glob, or any other tools.
