## Available Tools

You have no specialized code intelligence MCP tools. Use Claude's built-in tools to investigate the codebase:
- `Read` — read a file by path
- `Grep` — search file contents with regex patterns
- `Glob` — find files by pattern
- `Bash` — run shell commands

You may use any combination of these tools. There are no restrictions on which built-in tools you use.

Be thorough but efficient. The task requires accurate, specific answers (file names, line numbers, symbol names). Search and read as many files as needed to construct a complete answer.
